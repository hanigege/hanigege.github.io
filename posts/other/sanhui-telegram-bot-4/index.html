出国自建voip(二）创建 Telegram 机器人

要通过 Telegram 机器人 对接 三汇语音网关（SMG4000-C1LC） 的短信功能，您需要完成以下几个步骤：

步骤 1：

打开 Telegram，搜索并与 BotFather 聊天。

发送命令 /newbot 来创建一个新的机器人。

给机器人起个名字并设置用户名。

BotFather 会给你一个 API Token，用来与 Telegram 机器人进行交互。保存好这个 Token。

步骤 2：配置三汇网关发送短信 三汇语音网关一般支持通过 HTTP API 或 SMTP 发送短信。首先，确保三汇网关的短信服务已经启用，并且能够通过接口发送短信。

步骤 3：编写 Telegram 机器人和三汇网关的对接脚本 （本站有提供下载） 你需要编写一个脚本，使用 Telegram 机器人获取信息，并通过 HTTP 请求将短信发送到三汇网关。

步骤 3.1：安装 Telegram Bot 库（例如 Python 的 python-telegram-bot）（详情，请看voip出国系列三）
