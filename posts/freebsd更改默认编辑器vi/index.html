Freebsd更改默认编辑器vi

前提是新编辑器已经安装

FreeBSD 下你可以通过设置环境变量 EDITOR 来更改 crontab -e 使用的默认编辑器。下面是几种常见方法：

✅ 临时更改（当前 shell 有效）

EDITOR=nano crontab -e

或者用其他你喜欢的编辑器，如：

EDITOR=micro crontab -e
EDITOR=ee crontab -e
EDITOR=vim crontab -e

✅ 永久更改（加到 shell 启动文件）

如果你使用的是 sh 或 bash，可以将以下内容加到 ~/.profile 或 ~/.bashrc或 ~/bash_profile：

export EDITOR=nano

如果你使用的是 tcsh（FreeBSD 默认），请加到 ~/.cshrc 文件中：

setenv EDITOR nano

然后执行：

source ~/.cshrc

📦 补充提示：安装简易编辑器

如果你想用 nano 但系统没有，可以用 pkg 安装：

sudo pkg install nano
