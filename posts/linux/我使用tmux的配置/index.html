我使用tmux的配置

tmux.conf的配置

直接导入即可：

cat << 'EOF' > ~/.tmux.conf
# --- 基础设置 ---
set -g mouse off              # 关闭鼠标接管，恢复 iTerm2 原生选取和粘贴
set -g base-index 1
set -g pane-base-index 1
set -g renumber-windows on

# --- 状态栏美化 (通用无乱码版) ---
set -g status-style "bg=#282c34,fg=#abb2bf"
set -g status-left-length 30
set -g status-left "#[bg=#61afef,fg=#282c34,bold] #S #[bg=default,fg=#61afef] "

# 1. 开启 vi 风格的复制模式
setw -g mode-keys vi

# 2. 绑定 y 键为复制（并同步到系统剪贴板）
# 注意：tmux 3.2+ 版本配合 set-clipboard on 效果最佳
bind-key -T copy-mode-vi y send-keys -X copy-selection-and-cancel

# 3. 确保开启剪贴板透传支持
set -s set-clipboard on

# 4. 的响应速度10 毫秒、回滚的最大行数为 100 行、和支持 256 色终端
set-option -sg escape-time 10
set-option -g history-limit 100
set-option -g default-terminal "screen-256color"


# tmux 重新读取配置文件 Ctrl + a 然后按 r 即可
bind r source-file ~/.tmux.conf \; display-message "Config reloaded!"

# --- 快捷键优化 ---
# 把前缀键从 Ctrl+b 改成 Ctrl+a (更顺手)
unbind C-b
set -g prefix C-a
bind C-a send-prefix
EOF

导入后，退出终端重新登录生效

说明
优化美化底部栏：不隐藏底边栏，禁用tmux接管鼠标滚轮
其它优化功能看配置文件里的注释
