终端编辑器Micro教程，适合mac用户

Micro的介绍

在 Mac 上通过终端连接远端 Ubuntu 使用 Micro 编辑器是一个非常棒的选择。Micro 的设计初衷就是为了让习惯 GUI（如 VS Code）的用户在终端里也能拥有几乎一致的操作体验。

以下是一套完整的从安装到深度优化的教程，旨在让你的 Micro 像原生编辑器一样好用。

安装 (在 Ubuntu 远端)

  • 官方脚本（推荐，版本最新）
curl https://getmic.ro | bash
sudo mv micro /usr/local/bin/
  • 或者使用 APT：
sudo apt update && sudo apt install micro

核心配置：让它更好用

micro ~/.config/micro/settings.json
  • 添加以下内容：
{
    "colorscheme": "simple",
    "mouse": true,
    "saveundo": true,
    "tabsize": 4,
    "tabstospaces": true,
    "autoindent": true,
    "cursorline": true,
    "ruler": true,
    "clipboard": "terminal"
}

设置快捷键：删除和全局替换

micro ~/.config/micro/bindings.json
  • 添加以下内容：
{
    "Ctrl-k": "DeleteLine",
    "Ctrl-w": "command-edit:replaceall "
}

添加alias(可选）

micro ~/.bashrc
  • 添加以下内容：
# 写入
alias mr='micro'
# 生效
. ~/.bashrc

Shift + 方向键 如果出现乱码

  • Ubuntu 24.04 默认可能无法正确解析 iTerm2 发送的高级按键序列。
  • 解决方法： 在远端 Ubuntu 终端输入以下命令
export TERM=xterm-256color

如果输入后 Shift + 方向键 恢复正常，可以将其永久写入你的环境变量中：

echo 'export TERM=xterm-256color' >> ~/.bashrc
source ~/.bashrc
